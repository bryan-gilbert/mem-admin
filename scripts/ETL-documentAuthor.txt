The document author was set on upload to match the user who did the upload. This behaviour changed July 2017.
We need to now clear everything in the document author field.
Run the following in mongo after PR#1 is migrated to production.

db.documents.updateMany({documentAuthor :{ $ne : null }},{ $unset: { documentAuthor: ""}})
db.documents.updateMany({documentAuthor :{ $eq : "" } },{ $unset: { documentAuthor: ""}})

Run this to be sure everything is clear:
db.documents.find({$and : [ { documentAuthor : {$ne : ""} },{ documentAuthor : {$ne : null}}]},{documentAuthor:1})
db.documents.find({documentAuthor :{ $ne : "" }},{documentAuthor:1})

To make a copy of the relevant parts of the documents collection for testing run this
db.documents.aggregate( [ { $project: {displayName:1, documentAuthor:1, processed: { $literal: false }}},{ $out: "docs" } ])

To perform the update on Mongo 2.6.9
db.docs.update( {$and : [ { documentAuthor : {$ne : ""} },{ documentAuthor : {$ne : null}}]}, { $unset: { documentAuthor: ""}}, {multi: true })
